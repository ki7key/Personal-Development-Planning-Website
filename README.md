# Personal-Development-Planning-Website

基于 Qt Quick 开发的个人成长规划系统，帮助用户记录、规划和追踪个人成长目标与计划。

📌 项目简介
本项目是一个桌面端个人成长规划工具，旨在为用户提供一个简洁直观的界面，用于制定个人发展计划、记录成长里程碑，助力用户高效管理自我提升过程。

✨ 核心特色与功能
- 界面可视化：采用 Qt Quick 实现的现代化 UI，交互流畅
- 本地数据管理：项目文件本地存储，无需依赖外部数据库
- 1.每学期所修课程
- 2.参加的竞赛、项目、实习
- 3.成绩和GPA计算
- 4.设置目标预估分，并在实际分数出来后与实际分数进行对比
- 5.导出完整简历
- 6.智能建议系统（输入你的目标，比如留学/保研/考研，它可以整体帮你个性化策划需要加强哪一方面，以及具体加强方案）

🛠️ 技术栈
- 开发语言：C++
- 框架：Qt 6 / Qt Quick (QML)
- 构建工具：qmake
- 版本控制：Git & GitHub

🚀 如何运行
1.  环境准备
    - 安装 Qt 6.x 开发环境（含 Qt Quick 模块）
    - 配置 C++ 编译工具链（如 MinGW 或 MSVC）

2.  项目构建
    -克隆仓库：git clone git@github.com:ki7key/Personal-Development-Planning-Website.git
    -进入项目目录：cd Personal-Development-Planning-Website
    -使用 Qt Creator 打开 .pro 文件，构建并运行项目

🎨 界面结构设计

| 左侧区域 | 右侧区域 |
| :--- | :--- |
